Session management

Most API called require a 'token'. The only way to get one is to log in. Users can log in from multiple devices the same time, and tokens currently last forever, or until

  1. To log in, HTTP POST to /api/1/login with parameters username and password.
    username = , password = ,
    Results look like: 
    {
    "status": "OK", 
    "firstname": "Joe",
    "email": "bloggo@chef.com", 
    "lastname": "Bloggs", 
    "token": "68f22a73a6884dd7856f64160790e7e4"
}
    or like this: 
    {
    "status": "FAIL"
}
  2. To log out, HTTP POST to /api/1/logout with token obtained by logging in.
    token = ,
    Results look like: 
    {
    "status": "OK"
}
    or like this: 
    {
    "status": "FAIL"
}
